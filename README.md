# AI Resume Analyzer

A smart resume analysis web application built using Flask and Python that analyzes resumes, detects skills, provides recommendations, and suggests suitable internship roles.

## Features
✅ Resume Skill Analysis  
✅ Missing Skills Detection  
✅ Personalized Skill Recommendations  
✅ Resume Score with Dynamic Colors  
✅ Internship Role Prediction  

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Suggested Roles
- Data Analyst Intern
- Web Developer Intern
- AIML Intern
- General Software Intern

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
